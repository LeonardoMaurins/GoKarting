# GoKarting
TUDublin Project for Matt Smith using PHP with Symfony and Twig to create a website for Go Karting and information about races.

### Prerequisites

* [PHP](https://www.php.net/) - Scripting Language
* [Composer](https://www.npmjs.com/) - Package Manager

### Running locally

1. Navigate to the project directory within a terminal.
2. Execute ```composer install``` for the project vendor.
3. Execute ```php bin\console doctrine:fixtures:load``` to created pre-defined tables.
4. Execute ```php -S localhost:8000 -t ./public``` to host the website.
5. Navigate to the localhost on your browser at ```localhost:8000```.

## Features

* Login
* User, Staff & Admin Roles
* Information Pages
* Interactive race and user tables
* CRUD Implementation
* Different roles with certain privileges
* Codeception Tests
